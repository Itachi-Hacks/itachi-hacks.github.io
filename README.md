## 👤 About Me

I'm **Abhay Pandya**, a passionate and self-driven cybersecurity professional from **Pune, India**, specializing in **Vulnerability Assessment & Penetration Testing (VAPT)**, **Red Teaming**, and **Ethical Hacking**.

I don't just study security — I build it, break it, and document it. From developing my own offensive security tools to competing in live Bug Bounty championships, I approach cybersecurity with the mindset of an attacker and the discipline of a defender.

Currently working as a **Red Team Intern at CyArt (Ahmedabad)**, where I conduct adversarial simulations, identify vulnerabilities in real environments, and build automation tools for VAPT workflows using MCP Server architecture on Kali Linux.

I hold the **CIOSE (Certified Information and Offensive Security Expert)** certification from Demmisto Technologies — one of the most hands-on offensive security credentials in the program — and have completed **10+ industry certifications** across Ethical Hacking, Network Security, and Red Team Operations from globally recognized institutions including Cisco and EC-Council.

My work is not confined to labs. I have built and published **4 open-source security tools** on GitHub used for real-world ethical hacking and training, and I won **1st Prize at the Live Bug Bounty Championship 2026** organized by Demmisto Technologies in collaboration with Apollo Institute of Engineering and Technology.

I believe in learning by doing — every tool I build, every lab I break, and every vulnerability I find makes me sharper. If you are looking for someone who is deeply committed to the field of offensive security and brings both technical skill and creative problem-solving to the table, let's connect.
